## ⚡ COMO USAR ESTE LAB

Clone o repositório:

```bash
git clone [https://github.com/VagnerBellacosa/LaboratorioPython](https://github.com/VagnerBellacosa/LaboratorioPython)
cd LaboratorioPython
```

Execute qualquer missão:

```
python desafio_nome.py
```

Dica de veterano:
 👉 Leia o problema antes de ver a solução
 👉 Tente resolver sozinho primeiro
 👉 Depois compare com o código
